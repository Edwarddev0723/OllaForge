## ðŸ“– Overview

OllaForge is a CLI tool for generating and augmenting LLM training datasets using local Ollama models.

### Key Features

1. **Dataset Generation** - Create new datasets from natural language descriptions
2. **Dataset Augmentation** - Enhance existing datasets with AI-powered transformations
3. **Multiple Formats** - Support for SFT, Pre-training, Conversation, and DPO formats
4. **Quality Control** - Built-in QC for Traditional Chinese (Taiwan) datasets
5. **High Performance** - Concurrent processing with structured JSON output
