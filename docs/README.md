## ğŸ“– Overview

OllaForge is a CLI tool for generating and augmenting LLM training datasets using local Ollama models.

### Key Features

1. **Dataset Generation** - Create new datasets from natural language descriptions
2. **Dataset Augmentation** - Enhance existing datasets with AI-powered transformations
3. **Web Interface** - Browser-based UI for easy dataset management
4. **Multiple Formats** - Support for SFT, Pre-training, Conversation, and DPO formats
5. **Quality Control** - Built-in QC for Traditional Chinese (Taiwan) datasets
6. **High Performance** - Concurrent processing with structured JSON output
