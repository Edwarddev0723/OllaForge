# VS Code Configuration

This directory contains recommended VS Code settings for OllaForge development.

## Setup

Copy this directory to `.vscode` in your workspace:

```bash
cp -r .vscode.example .vscode
```

## Contents

- `settings.json` - Workspace settings for Python development
- `extensions.json` - Recommended extensions

## Recommended Extensions

- **Python** - Python language support
- **Black Formatter** - Code formatting
- **Ruff** - Fast Python linter
- **Mypy Type Checker** - Type checking
- **Even Better TOML** - TOML file support
